# Security Policy / מדיניות אבטחה

## Supported Versions
The `main` branch is the actively supported version.

## Reporting a Vulnerability
Please do not open public issues for sensitive vulnerabilities.

Report privately by opening a GitHub Security Advisory or contacting the maintainer directly.
Include:
- Steps to reproduce
- Impact assessment
- Suggested mitigation (if available)

## Response Targets
- Initial acknowledgment: within 7 days
- Triage/update: within 14 days

## דיווח חולשות (עברית)
נא לא לפתוח Issue פומבי עבור חולשות אבטחה רגישות.
יש לדווח באופן פרטי (Security Advisory או פנייה ישירה למתחזק).
יש לצרף:
- שלבי שחזור
- הערכת השפעה
- הצעת תיקון (אם קיימת)
